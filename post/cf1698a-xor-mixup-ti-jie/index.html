CF1698A XOR Mixup 题解

2022-07-02

题意简述

给定 n−1n-1n−1 个数，其中有 n−1n-1n−1 个数属于数列 aaa，剩下一个数为 xxx。

xxx 为 a1⊕a2⊕⋅⋅⋅⊕an−1a_1 \oplus a_2 \oplus \cdot \cdot \cdot \oplus a_{n-1}a1​⊕a2​⊕⋅⋅⋅⊕an−1​（$\oplus $ 表示异或）。

请求出 xxx 的值（输出其中一个）。

解题思路

我们根据题意可以按如下推导：

a1⊕a2⊕⋅⋅⋅⊕an−1⊕xa_1 \oplus a_2 \oplus \cdot \cdot \cdot \oplus a_{n-1} \oplus x a1​⊕a2​⊕⋅⋅⋅⊕an−1​⊕x

(a1⊕a2⊕⋅⋅⋅⊕an−1)⊕x(a_1 \oplus a_2 \oplus \cdot \cdot \cdot \oplus a_{n-1}) \oplus x (a1​⊕a2​⊕⋅⋅⋅⊕an−1​)⊕x

(a1⊕a2⊕⋅⋅⋅⊕an−1)⊕(a1⊕a2⊕⋅⋅⋅⊕an−1)(a_1 \oplus a_2 \oplus \cdot \cdot \cdot \oplus a_{n-1}) \oplus (a_1 \oplus a_2 \oplus \cdot \cdot \cdot \oplus a_{n-1}) (a1​⊕a2​⊕⋅⋅⋅⊕an−1​)⊕(a1​⊕a2​⊕⋅⋅⋅⊕an−1​)

∴a1⊕a2⊕⋅⋅⋅⊕an−1⊕x=0\therefore a_1 \oplus a_2 \oplus \cdot \cdot \cdot \oplus a_{n-1} \oplus x = 0 ∴a1​⊕a2​⊕⋅⋅⋅⊕an−1​⊕x=0

所以这 nnn 个数中间任意一个数都可以是 xxx。这里我用了 rand() 来做，所以要用 srand(time(NULL)) 初始化一下。

代码实现

#include<bits/stdc++.h>
using namespace std;
int t,n,a[1000005];
int main() {
	scanf("%d",&t);
	while(t--) {
		scanf("%d",&n); srand(time(NULL));
		for(int i=1;i<=n;i++) scanf("%d",&a[i]);
		printf("%d\n",a[rand()%n]);
	}
	return 0;
}
